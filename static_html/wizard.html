Wizard List

Set your IP data

To set up a hosted URL, fill out this form. Rex uses the same URL to set and view public IP sets, so the "Set successfully" page is the page where people can go to to see your current connection information (and for that reason, if you try to refresh the page, the browser will try to resend the information, loading the page in POST mode, so you need to click in your address bar and hit enter to get it to load in GET mode, which displays your info. This only affects you if you just set your information, and will not affect other people viewing the page).

Reserve a URL

It's not cool if just anyone can set your IP data to anything they want. In fact that's terrible. While we're working to provide more security than just password protection (SSL access and RSA cryptography), we're also in a rush to get this thing out the door now that internet censorship is getting real in the U.S. That's a squeezy yet ill-defined deadline. Anyways, to reserve a named URL as your own and prevent other people from using it, set it up here.
