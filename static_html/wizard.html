Wizard List

To set up a hosted URL, fill out this form. Rex uses the same URL to set and view public IP sets, so the "Set successfully" page is the page where people can go to to see your current connection information (and for that reason, if you try to refresh the page, the browser will try to resend the information, loading the page in POST mode, so you need to click in your address bar and hit enter to get it to load in GET mode, which displays your info. This only affects you if you just set your information, and will not affect other people viewing the page). Name reservation is not a feature yet but will be eventually.
